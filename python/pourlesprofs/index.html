Une introduction à Python pour les Math au lycée

Parti Pris !

Dans ce cours j'ai volontairement omis de parler des fonctions print et input et ce pour plusieurs raisons :

  • enseigner leur usage fait perdre beaucoup de temps alors que ce n'est pas ce qui intéresse le plus le mathématicien dans l'usage d'un langage de programmation
  • éviter la confution que les élèves font entre print et return
  • expliquer pourquoi on doit écrire int(input("blabla")) n'est pas si simple et perd les élèves dans des détails qui ne les concernent pas
  • le plus important est l'utilisation de fonction et justement on peut facilement faire le parallèle avec les fonctions vues en maths. On a une ou des données en entrée et un ou des résultats en sortie
  • quand les élèves doivent modéliser un problème à l'aide d'un programme il perde beaucoup de temps à écrire les questions int(inpiut("Quelle est l'abscisse du point A ?")) et pendant ce temps ils n'ont toujours pas écrit le programme
  • les élèves n'ont pas vocations à devenir informaticiens grace au cours de Maths. Il y a NSI pour cela.

Comment utiliser ce cours ?

Ce site s'adresse principalement à des enseignants de Mathématiques qui ont déjà quelques vagues notions du langage Python. Vous n'y trouverez pas vraiment de cours, mais quelques explications succintes. Vous trouverez un ensemble d'exemples de ce que l'on peut proposer pour introduire le langage Python au lycée, principalement au niveau seconde. J'ai essayé de proposer des exercices "intéressants" qui s'éloignent le plus possible de ce que l'on voit souvent comme ce genre de programme qui n'offre que très peu d'intérêt mathématique (et algorithmique). 

    """
    Voici un mauvais exemple !
    """
    age = int(input("Quel est ton âge ?"))
    if age < 18 :
        print("tu es mineur.e")
    else :
        print("tu es majeur.e")

Pour ce cours, j'utilise des liens vers le site Basthon qui est un environnement de programmation pour Python qui ne nécessite aucune installation au préalable et qui permet de faire du Python directement sur son navigateur. En outre, Basthon fonctionne parfaitement sur les smartphones et les tablettes. Enfin l'environnement De Basthon est celui qui est repris sur Capytale.

Pour chaque exercices, un lien vers Basthon sera donné pour effectuer le programme et un autre pour voir une correction possible.

Ce cours est composé des quatre parties suivantes :

  1. Fonction
  2. Structure conditionnelle
  3. Boucle bornée (boucle POUR)
  4. Boucle non bornée (boucle TANT QUE)
