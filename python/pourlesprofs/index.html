Une introduction à Python pour les Maths au lycée

Parti Pris !

Dans ce cours j'ai volontairement omis de parler des fonctions print et input et ce pour plusieurs raisons :

  • enseigner leur usage fait perdre beaucoup de temps alors que ce n'est pas ce qui intéresse le plus le mathématicien dans l'utilisation d'un langage de programmation;
  • éviter la confution que les élèves font entre print et return;
  • expliquer pourquoi on doit écrire int(input("blabla")) n'est pas si simple et perd les élèves dans des détails qui ne les concernent pas;
  • le plus important est l'utilisation de fonction et justement on peut facilement faire le parallèle avec les fonctions vues en maths. On a une ou des données en entrée et un ou des résultats en sortie;
  • quand les élèves doivent modéliser un problème à l'aide du langage de programmation Python, ils perdent beaucoup de temps à écrire les questions int(input("Quelle est l'abscisse du point A ?")) et pendant ce temps ils n'ont toujours pas écrit la réelle structure algorithmique du programme;
  • les élèves n'ont pas vocations à devenir informaticiens grace au cours de Maths. Si besoin, il y a NSI pour cela à partir de la première.

Comment utiliser ce cours ?

Ce site s'adresse principalement à des enseignants de Mathématiques qui ont déjà quelques vagues notions du langage Python. Vous n'y trouverez pas vraiment de cours, mais quelques explications succintes. Vous trouverez un ensemble d'exemples de ce que l'on peut proposer pour introduire le langage Python au lycée, principalement au niveau seconde. J'ai essayé de proposer des exercices "intéressants" qui s'éloignent le plus possible de ce que l'on voit souvent comme ce genre de programme qui n'offre que très peu d'intérêt mathématique (et algorithmique) mais qui se rapprochent le plus possible des activités que l'on fait déjà en classe, afin de montrer comment l'on peut intégrer la programmation en Python en classe de seconde. 

    """
    Voici un mauvais exemple !
    """
    age = int(input("Quel est ton âge ?"))
    if age < 18 :
        print("tu es mineur.e")
    else :
        print("tu es majeur.e")

Basthon

Pour ce cours, j'utilise des liens vers le site Basthon qui est un environnement de programmation pour Python qui ne nécessite aucune installation au préalable et qui permet de faire du Python directement sur son navigateur. En outre, Basthon fonctionne parfaitement sur les smartphones et les tablettes. Enfin l'environnement de Basthon est celui qui est repris sur Capytale, disponible pour les élèves et les enseignants à partir de l'ENT (espace numérique de travail) des établissements des Pays de la Loire, e-lyco.

Pour chaque exercice, un lien vers Basthon sera donné pour effectuer le programme et un autre pour voir une des corrections possibles.

Le cours

Ce cours est composé des quatre parties suivantes :

  1. Fonction
  2. Structure conditionnelle
  3. Boucle bornée (boucle POUR)
  4. Boucle non bornée (boucle TANT QUE)
