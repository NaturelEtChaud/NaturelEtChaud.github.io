Les variables

Pour commencer à prendre en main le langage Python, on peut effectuer quelques petites manipulations autour des variables. Il ne me semble pas utile en mathémtiques de parler des différents types de variables autre que les entiers int et les nombres décimaux (ou nombres à virgule flottante ou encore flottant) float. Les chaines de caractères sont inutiles puisque je n'utilise ni la fonction print ni la fonction input. On pourra aller plus loin sur la notion de flottants en suivant ce lien.
En ce qui concerne le travail sur les variables le plus simple à mon sens est de leur proposer des programmes de calcul comme ceux déjà vus au collège. On pourra aussi dès le départ, montrer l'intérêt de d'utiliser des fonctions.
Pour rappel, les différentes opérations possibles pour les nombres sont les suivantes :

Opération Traduction en Python Opération Traduction en Python
Somme de a et de b a + b Différence de a par b a - b
Produit de a et de b a * b Quotient décimal de a par b a / b
Quotient entier de a par b a // b Reste du quotient entier de a par b (modulo) a % b
a exposant b a ** b racine carrée de a sans passer par la fonction racine du module math a ** 0.5

Des exercices

Exercice 1

[D'après le Brevet du 14 juin, Centres étrangers groupe 1]
Amir et Sonia ont chacun inventé un programme de calcul.

Programme d'Amir :
  • Choisir un nombre
  • Soustraire 5
  • Prendre le double du résultat
 Programme de Sonia :
  • Choisir un nombre
  • Ajouter 3
  • Multiplier le résultat par le nombre choisi
  • Soustraire 16
  1. Montrer que si le nombre choisi au départ est 6 alors on obtient 2 avec le programme d’Amir et on obtient 38 avec celui de Sonia.
  2. Vérifier qu'avec le nombre de départ 2, les programmes d'Amir et de Sonia donnent le même résultat.
  3. Existe-t-il d'autres nombre de départ qui donnent le même résultat avec ces deux programmes de calcul.
Lien Basthon pour programmer Lien pour la correction
Aide
Attention à bien utiliser le mot clef return et non pas la fonction print si vous définissez une fonction. Vous en saurez plus dans la partie suivante à ce sujet.

Exercice 2

Quelles seront les valeurs des variables a, b et éventuellement c après exécution des affectations suivantes ? 

  1.         a = 2
        b = a - 3
        a = 3
    Réponse
    a = 3 et b = -1 
  2.         a = 15
        b = - 8
        c = a + b
        a = 6
        c = b - a
    Réponse
    a = 6, b = -8 et c = -14 
  3.         a = 15
        b = - 8
        b = a + b
        a = 6
        c = b - a
    Réponse
    a = 6, b = 7 et c = 1 
  4.         a = 5
        b = 2
        b = a
        a = b
    Réponse
    a = 5 et b = 5
    Malheuresement les variables a et b n'ont pas été échangées ...
