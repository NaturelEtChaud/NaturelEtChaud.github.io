La Fonction

Une fonction en Python se comporte comme une fonction en Mathématiques. On la définit dans un premier temps comme on peut définir une fonction par son expression mathématique. Ensuite on utilise cette fonction en lui donnant des informations en entrée pour en récupérer des réponses en sorties.
Pour créer une fonction, on commence par le mot clef def, on donne ensuite le nom de la fonction suivi de parenthèses dans lesquels on donne le ou les variables corerspondants aux valeurs que l'on donnera en entrée. Cette première ligne est toujours terminée par deux points :.
Après ces deux points on valide la ligne avec la touche Entrée. Le curseur se place directement à la ligne suivante avec une indentation. Chaque instruction qui définit la fonction doit garder au minimum cette indentation. On peut également l'obtenir avec la touche de tabulation tab mais surtout pas (!!!) en utilisant plusieurs fois la touche espace (cette pratique est une source récurrente d'erreurs !)
Une fonction se termine par le mot clef return pour préciser ce qui est renvoyé par la fonction. 
    def maFonction(entree1, entree2, ...):
        instruction1
        instruction2
        instruction3
        ...
        return valeur1, valeur2, ...

Exemple

La fonction mathématique d'expression f(x) = 1 x se définit en Python par : 

    def f(x):
        calcul = 1 / x
        return calcul
ou mieux par 
    def f(x):
        return 1 / x
Une fois la fonction définie et exécutée, on peut l'utiliser pour calculer dans la console les valeurs des images voulues comme par exemple f(2), f(-10), f(0), ... . 

    >>> f(2)
    0.5
    >>> f(-10)
    -0.1
    >>> f(0)
    Traceback (most recent call last):
      File "<input>", line 1, in <module>
      File "<input>", line 2, in f
    ZeroDivisionError: division by zero
On constate dans cet exemple que Python nous informe qu'il ne peut pas diviser par zéro.

Programmer des fonctions pour calculer les aires d'un carré, d'un rectangle, d'un triangle ou d'un trapèze.

Lien Basthon pour programmer Lien pour la correction
Aide
L'exposant s'écrit ** en Python, ainsi x2 s'écrit x ** 2.

Exercice 3

  1. Programmer une fonction qui calcule dans un repère les coordoonées du milieu de deux points connaissant leurs coordonnées.
  2. Programmer une fonction qui calcule dans un repère les coordonnées d'un vecteur défini par deux points connaissant leurs coordonnées.
  3. Programmer une fonction qui calcule dans un repère le déterminant de deux vecteurs connaissant leurs coordonnées.
  4. Programmer une fonction qui calcule dans un repère orthonormée la distance entre deux points connaissant leurs coordonnées.
Lien Basthon pour programmer Lien pour la correction
Aide 1
Quelles sont les entrées ? Quelles sont les sorties ?
Aide 2

Aide 3
Pour la dernière question, on a besoin de calculer la racine carrée d'un nombre. Il faut tout d'abord importer la fonction sqrt (pour SQuare RooT) du module math avec l'instruction from math import sqrt. On peut ensuite calculer la racine carrée de n'importe quel nombre sqrt(x).
