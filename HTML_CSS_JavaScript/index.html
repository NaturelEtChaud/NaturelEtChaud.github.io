II- Pour les premières NSI

a) Les formulaires

  • Voici quelques exemples de formulaires écrits en HTML : formulaires

Pour envoyer les données du formulaire au serveur, il existe deux méthodes :

  • la méthode GET :
    Les données du formulaire sont inscrites dans l'url est sont envoyés au serveur.
    C'est une méthode très pratique mais peu confidentielle.
  • la méthode POST :
    Les données du formulaire ne sont plus inscrites dans l'url mais elles sont envoyés dans le corps de la requête via le protocole HTTP.
    C'est une méthode est très confidentielle ;-)
